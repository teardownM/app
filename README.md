# TeardownM Client

This is the server browser for TeardownM.

<br />

## Getting Started

|         |                    |
| ------- | ------------------ |
| Install | · `yarn install`   |
| Develop | · `yarn run dev`   |
| Build   | · `yarn run build` |

<br />
